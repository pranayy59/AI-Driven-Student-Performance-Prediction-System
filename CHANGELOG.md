# Changelog

All notable changes to this project are documented here.

## [1.1.0] - 2026-07-15

### Added

- Professional Streamlit dashboard with sidebar navigation.
- Prediction probability, confidence level, risk assessment, and recommendation.
- Downloadable prediction CSV.
- Precision, recall, F1 score, accuracy, and ROC-AUC reporting.
- Precision-recall curve and model accuracy comparison visualizations.
- Project report, contributing guide, code of conduct, and changelog.

### Improved

- Refined preprocessing with missing-value imputers.
- Improved model metadata for dashboard and reproducibility.
- Rewrote README for portfolio presentation.
- Strengthened error handling in data loading and training.

## [1.0.0] - 2026-07-15

### Added

- Initial machine learning pipeline.
- CLI prediction script.
- Streamlit prediction app.
- EDA visualizations.
- Reproducible notebook.
